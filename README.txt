###УСТАНОВКА

Написано с использованием Python 3.8, возможно, запускается и на более ранних версиях

Для работы необходимы доп. модули, указанные в файле requirements.txt

Для их автоматической установки необходимо выполнить

$ pip3 install -r requirements.txt

или

$ python3.8 -m pip install -r requirements.txt

###Использование

$ script.py <file> <vnmin> <vnmax> <tmin> <tmax> <dt>

Все аргументы являются обязательными

file - входной файл со спектром. Формат - линия чисел, разделённых запятой.
vnmin, vnmax - предполагаемый диапазон волновых чисел спектра
tmin, tmax - предполагаемый температурный диапазон, К
dt - шаг по температуре, с которым необходимо считать подгоночные спектры

Во время работы программа создаёт в рабочей директории поддиректорию "tmp", в которой хранятся данные из базы HITRAN.
В конце работы программа визуализирует данный ей и симулированный спектры, остатки, рассчитаную температуру.
